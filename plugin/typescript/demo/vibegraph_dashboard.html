Recording Flow

1. Register Profiles

Alice and Bob create onchain social identities with unique handles.

2. Give Vibe

Alice signs a reputation attestation for Bob with a tag, note, and score.

3. Confirm RPC

The demo confirms tx inclusion through Canopy RPC and prints tx hashes.

Run this in another terminal: npm run demo:vibegraph
